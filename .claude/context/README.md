# Project Context - TL Board Game

Unreal Engine recreation of a board game.

## Folders

- **Assets/** - Asset descriptions and usage information
- **Content/** - Blueprint and asset definitions for project implementations
- **Development/** - C++ class definitions, functionality, state, events, purpose, coding standards
- **Gameplay/** - Game entities and mechanics (base for vertical slice module separation)
- **Scenarios/** - Game flows, states, and events
- **Tests/** - Flow tests for development validation
- **Tools/** - Definitions for AI interaction with Unreal Engine (Blueprint generation, etc.)
