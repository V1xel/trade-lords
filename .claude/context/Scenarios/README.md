# Scenarios

Scenarios demonstrate how game systems interact. Each shows mechanics in action.

## Available Scenarios

### Game Flow
- **CompleteRound.md** - Full round cycle (action → production → scoring)

### Production
- **ProductionChain.md** - Grain to Whisky chain with Godwyn aging
- **ContractFulfillment.md** - Gathering and delivering contract goods

### Lord Abilities
- **LordAbilityCombo.md** - Bertram factories, Aldric dual contracts, Elspeth milk sales

### Trading
- **MarketDynamics.md** - Price changes, Isolde trading, merchant limits

### Expansion
- **ShippingExpansion.md** - River crossing, loch crossing, Marina deltas, Caspian fishermen

## Reading Scenarios

Each scenario shows:
- Initial state
- Action sequence
- Game state changes
- Mechanical interactions

Focus on how systems connect, not optimal strategy.
