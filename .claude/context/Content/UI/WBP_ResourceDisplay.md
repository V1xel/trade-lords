# WBP_ResourceDisplay

**Type:** Widget Blueprint (UUserWidget)

**Description:**
Compact resource display widget showing all player resources.

**Components:**
- Resource grid (icon + amount)
- Wool, Milk, Grain (basic goods row)
- Cheese, Bread, Whisky, Beer (processed goods row)
- Cotton, Tobacco, Sugar Cane, Hops (import goods row)
