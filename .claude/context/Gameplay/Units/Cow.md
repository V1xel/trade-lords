# Cow

## Overview

Cows are production units that produce Milk. Milk can be used directly or processed into Cheese for higher value.

## Properties

**Type**: Production Unit
**Cost**: £6
**Terrain**: Grassland only
**Production**: 1 Milk per Production Phase

## Placement

### Requirements
- Grassland terrain hex
- Adjacent to owned unit OR shipping range
- Pay £6 (unit) + terrain cost

### Examples
- Cow on £2 grassland = £8 total
- Cow on £5 mixed grassland = £11 total

## Production

### Production Phase
- Each Cow produces 1 Milk
- Automatic (no input required)
- Milk added to player's stock

### Processing Option
- Milk can be used directly
- OR processed: Milk → Cheese Dairy → Cheese
- Cheese worth more than Milk

## Strategic Use

### Dairy Chain Strategy
- Cows + Cheese Dairies
- Milk → Cheese conversion
- Higher value output

### Direct Use
- Sell Milk on market
- Use Milk for contracts
- Dame Elspeth sells for flat £8

## Lord Synergies

### Dame Elspeth (Strong)
- Starts with 2 Cows + 1 Cheese Dairy
- Can sell Milk for £8 (guaranteed)
- Dairy specialist

### Sir Bertram
- Cheap Cheese Dairies (£3-£5 discount)
- Efficient dairy chain

## Processing Chain

### Basic Milk Use
```
Cow → Milk (use directly)
```

### Cheese Production
```
Cow → Milk → Cheese Dairy → Cheese
```
