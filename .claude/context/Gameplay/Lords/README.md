# Lords Module

## Overview

Each player controls one lord with a unique ability. Lords provide asymmetric gameplay while maintaining balanced competitive options.

## The 9 Lords

1. **Lord Aldric the Negotiator** - Contract specialist
2. **Master Godwyn the Distiller** - Whisky aging expert
3. **Lady Isolde the Merchant Princess** - Trading master
4. **Sir Bertram the Builder** - Factory efficiency
5. **Dame Elspeth the Shepherdess** - Dairy production
6. **Baron Roderick the Foreman** - Worker legion
7. **Lord Caspian the Steward** - Flexible workers
8. **Lady Marina of the Estuaries** - River delta specialist
9. **Brewmaster Oswald the Bold** - Beer production

## Lord Selection

**Timing**: During game setup, before first round

**Process**:
1. Randomly draw (number of players + 1) lords
2. Players choose in reverse turn order
3. Each lord chosen once

**Game State**:
- Available lords pool
- Player turn order
- Selected lord per player

## Common Properties

### Identity
- Name
- Title
- House colors (primary/secondary)

### Starting Resources
Each lord begins with unique starting resources:
- Gold amount (£6-£12)
- Initial units (1-3 units)
- Special bonuses (merchants, shipping levels)

### Ability Types

**Passive**: Always active
- Example: Sir Bertram's factory cost reduction

**Production Phase Triggered**: Activates during production
- Example: Master Godwyn's whisky aging

**Action Triggered**: Activates when player takes specific action
- Example: Lady Isolde's trade bonus

**Active**: Player chooses when to use
- Example: Dame Elspeth's milk selling

## Balance Framework

### Power Curves
- **Early Game**: Baron Roderick, Lady Isolde (immediate advantages)
- **Mid Game**: Sir Bertram, Lady Marina (scaling bonuses)
- **Late Game**: Master Godwyn (compound growth)

### Complexity
- **Beginner**: Lady Isolde, Dame Elspeth, Baron Roderick
- **Intermediate**: Sir Bertram, Lady Marina, Brewmaster Oswald
- **Advanced**: Lord Aldric, Master Godwyn, Lord Caspian

### Strategy Archetypes
- **Economic**: Lady Isolde, Dame Elspeth
- **Production**: Sir Bertram, Master Godwyn, Brewmaster Oswald
- **Expansion**: Baron Roderick, Lady Marina, Lord Caspian
- **Flexible**: Lord Aldric

## Game States Affected by Lords

### Player State
- Gold amount
- Owned units
- Available merchants (2-6)
- Shipping level (0-5)
- Active contracts (1-2)
- Resource inventory
- Special trackers (e.g., Godwyn's aging cellar)

### Per-Turn State
- Actions taken
- Merchants used
- Production generated
- Abilities triggered

### Per-Round State
- Contracts taken
- Territory expansion
- Factory construction
