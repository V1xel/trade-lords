# Wool

## Overview

Wool is a basic good produced by Sheep. Simplest resource chain - no processing required.

## Properties

**Type**: Basic Good
**Produced By**: Sheep
**Processing**: None (used directly)
**End Game Value**: 1 VP per Wool

## Production

### Production Phase
- Each Sheep produces 1 Wool
- Automatic (no input needed)
- Wool added to player's stock

### No Processing Required
- Wool used directly for contracts and trading
- Simplest resource chain
- No factory investment needed

## Uses

### Export Contracts
- Wool required for certain contracts
- Direct use (no processing)
- Standard contract rewards

### Market Trading
- Can sell Wool for gold
- Can buy Wool (if needed for contracts)
- Market price varies

### End Game
- 1 VP per Wool in stock
- Lower value than processed goods (2 VP)

## Strategic Value

### Advantages
- Simple production (just Sheep)
- No processing buildings needed
- Immediate use
- Low investment

### Disadvantages
- Lower contract value than processed goods
- 1 VP end game (vs 2 VP for processed)
- No processing chain options
- Limited strategic depth

## Game States

### Player Stock
- `wool: number` (count in stock)

### Per Production Phase
- Count Sheep units
- Add that many Wool to stock

## Common Patterns

### Wool Focus
- 3-4 Sheep
- 3-4 Wool per round
- Simple economy
- Fulfill Wool contracts

### Diversified Production
- 1-2 Sheep alongside Cows/Fields
- Mixed resources
- Flexible contracts

## Economic Value

**Production Cost**:
- Sheep: £5 + terrain
- Produces 1 Wool per round

**Wool Market Value**: ~£5-£7 (varies)

**Wool Contract Value**: Standard rewards

## Strategy Decisions

### When to Produce Wool
- Simple economy desired
- Early game resources
- Wool contracts available
- Diversification

### When to Skip
- Focus on processed goods (higher value)
- Limited grassland (use for Cows/Fields)
- Wool contracts not valuable

- Starts at base price
- Rises when bought

- Creates market dynamics
- Price fluctuates
