# Claude Code Documentation

This project contains context and documentation for Claude Code in the `.claude/context` folder.

## Quick Start

Check the `.claude/context/` directory for detailed documentation:

- **README.md** - Overview of the context structure
- **Assets/** - Asset-related documentation
- **Content/** - Content and blueprint documentation
- **Development/** - Development guides and workflows
- **Gameplay/** - Gameplay systems documentation
- **Scenarios/** - Common scenarios and examples
- **Tests/** - Testing documentation
- **Tools/** - Tools and utilities documentation

## Additional Resources

The `.claude/` folder also contains:
- `commands/` - Custom slash commands
- `hooks/` - Event hooks configuration
- `skills/` - Custom skills
- `settings.json` - Claude Code settings
